> **Draft.** This tag does not ship until [`GO_NO_GO.md`](../docs/release/GO_NO_GO.md)

The first version a stranger can install with no founder in the

## What this is

Recall is a **local-first continuity operating system**. You get
interrupted; Recall reconstructs what you were doing and, when you
come back, offers the one investigation you can step straight into.
Everything stays on your machine — no cloud, no telemetry, no
account.

## Install

- **Windows 10/11 (x64):** download `Recall-Setup.exe`, double-click.
- **macOS:** Preview only — see [`MAC_BUILD_STATUS.md`](../docs/release/MAC_BUILD_STATUS.md).
- Full steps: [`INSTALL.md`](../docs/release/INSTALL.md).

## Highlights since 0.1.0

- **One-double-click Windows installer** — per-user, no admin,
  shortcuts, optional launch-on-login.
- **Redesigned launcher** — the recovery / investigation / trust
  card set (Phase 4K).
- **Productized browser extension** — a calm memory surface with
  install-aware pairing (Install / Open / Repair).
- **Sharper recovery** — fewer, stronger, evidence-specific
  recoveries; investigation grouping that keeps a file with the
  work it belongs to.

See [`CHANGELOG.md`](../docs/release/CHANGELOG.md) for the full phase-by-phase
record.

## Known limitations

- Installer is **unsigned** — SmartScreen warns on first run; verify
  the SHA-256 (below).
- macOS is **Preview** — no verified `.dmg` yet.
- Recovery biases toward **silence** — early on you may see nothing,
  and that is correct, not broken.

Full list: [`PUBLIC_ALPHA.md`](../docs/founder/PUBLIC_ALPHA.md) § known limitations.

## Verify your download

```
# Windows
Get-FileHash .\Recall-Setup.exe -Algorithm SHA256
```

Compare against `checksums/SHA256SUMS`, attached to this release.
