# Feedback

This file is yours. Write here.

There is **no form**, **no structure**, **no
length minimum**. One sentence is fine. A wall
of text is fine. Bullet points are fine. Voice
note transcribed by you is fine.

We will read every word.

---

## What's useful

Most useful, in order:

1. **Verbatim quotes** of what surprised you
   (good or bad). Don't paraphrase.
2. **Specific failures** with what you expected
   vs. what happened. Severity guess is helpful.
3. **Confusions** — moments where you didn't
   know what to do, or what was happening.
4. **What you wished was there.** Even if it's
   in the "we will not build" list, knowing you
   wanted it is data.
5. **What you stopped doing** because Recall
   replaced it.

Less useful:

- Generic "this is cool"
- Generic "this needs more polish"
- Feature requests that are really new products
  ("can it summarise my day?")

Three options, pick any:

| Channel               | When                                    |
|-----------------------|-----------------------------------------|
| **This file**         | Default. Edit it locally, paste later.  |
| **GitHub issue**      | Reproducible bug, or anything public-ok. |
| **Email founder**     | Sensitive / NDA-adjacent / private.     |

The file lives in your local clone. We do not
auto-scrape it. You explicitly send / paste it.

## Prompts (if you're stuck)

Pick one and answer in two lines:

- The first time you opened the launcher, the
  feeling was: _____
- The most surprising thing Recall did was: _____
- The most confusing thing Recall did was: _____
- I almost uninstalled when: _____
- I kept using it because: _____
- I would describe Recall to a friend as: _____
- The one thing I want it to do differently is: _____

## Template — for the founder's intake

If you want to be maximally useful, this is the
five-field template we like:

```
HANDLE:        alpha-007         (or whatever you registered as)
PLATFORM:      Windows 11 + Chrome
DAYS RUN:      4
INSTALL WENT:  smooth / hiccup / hard fail
FIRST OPEN:    when, what you saw, how it felt
RECOVERY:      saw one? clicked it? right work?
WOW:           one verbatim quote, or "none yet"
FAILURE:       one verbatim quote, or "none yet"
WOULD MISS:    yes / no / don't know
NEXT:          would you re-install after a reset? (y/n)
```

That's it.
