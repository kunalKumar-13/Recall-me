- TypeScript build (`npx tsc --noEmit`) — **clean**.
- Production bundle size unchanged (these packages were
  already tree-shaken by Next's webpack pipeline; the
  cleanup removes them from `node_modules` only).

## `apps/extension/ui/package.json`

### Before

```json
"dependencies": {
  "framer-motion": "^11.3.0",
  "playwright": "^1.48.0",   ← misplaced (build tool, not runtime)
  "react": "^18.3.1",
  "react-dom": "^18.3.1"
}
```

### After

```json
"dependencies": {
  "framer-motion": "^11.3.0",
  "react": "^18.3.1",
  "react-dom": "^18.3.1"
},
"devDependencies": {
  …,
  "playwright": "^1.48.0",   ← moved here
  …
}
```

### Impact

- `playwright` moved from `dependencies` →
  `devDependencies`. Consumer is
  `apps/extension/ui/capture_extension.mjs`, a build-time
  capture tool — it doesn't ship in the popup bundle.
- TypeScript build (`npx tsc --noEmit`) — **clean**.
- Production bundle (`npx vite build`) — **293 KB JS,
  identical** before and after. Confirms playwright was
  never in the popup bundle anyway; the move is cosmetic
  + correctness, no runtime change.
